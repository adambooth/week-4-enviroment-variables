Steps

Client:

1. Create a client folder

2. run npm create vite@latest

3. choose vanilla

4. choose javascript

5. choose no

6. choose yes to install and run

Server:

1. Create a server folder

2. run npm init -y

3. install pg --> npm i pg

4. install express --> npm i express

5. Create .gitignore and .env

6. add "type": "module" to package.json

7. add "dev": "node --watch ./server.js"

8. Look at each file to see how they are connected and work together
